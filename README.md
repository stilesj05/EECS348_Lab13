# EECS 348 – Lab 13: Debugging and Testing

This project contains a C++ program used to demonstrate the concepts of **fault**, **error state**, and **failure** in a simple attendance-checking function. The repository includes:

- `lab13.cpp` — the main program
- `Makefile` — build configuration for compiling and cleaning
- `Lab13` test cases and output (printed by the program)
